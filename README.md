# Tomasz ma dziś 25 (statyczna stronka)

Strona na 25. urodziny Tomasza: pasek życia do 75, przeliczniki "25 lat to...", rozsypane interaktywne fakty na tle, konfetti i muzyka z pliku MP3.

## Darmowy hosting (GitHub Pages)

1. Utwórz repo na GitHubie.
2. Wrzuć zawartość folderu `birthday-25/` do katalogu głównego repo.
3. Zrób commit i push na `main`.
4. Na GitHubie: Settings -> Pages
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
5. Po chwili pojawi się adres URL.

Uwaga: automatyczne odtwarzanie muzyki bywa blokowane przez przeglądarki. Na stronie jest awaryjny przycisk do odblokowania dźwięku.
